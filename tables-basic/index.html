What this example covers

The page covers the same core data-table concerns using Quark's table primitives.

Searching and sorting

TableSearch and sortable Th columns keep the table interactive without extra scaffolding.

Pagination

TableBottomBar, TableInfo, and TablePagination keep paging controls aligned.

Empty state

TableNoData gives the table a dedicated no-results state instead of collapsing the layout.

Server-side interactions

The demo fetches rows through OnInteraction so filtering and paging can scale beyond in-memory samples.
