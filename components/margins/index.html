Responsive Margins

OnBase, OnMd, OnLg, OnXl, On2xl.

Apply different margins based on device screen sizes for optimal responsive design.

OnBase

Portrait phones (< 576px)

Margin.OnBase.Is1
m-1

OnMobile

Landscape phones (≥ 576px)

Margin.OnMd.Is3
sm-m-3

OnTablet

Tablets (≥ 768px)

Margin.OnMd.Is2
md-m-2

OnLaptop

Laptops (≥ 992px)

Margin.OnLg.Is4
lg-m-4

OnDesktop

Desktops (≥ 1200px)

Margin.OnXl.Is3
xl-m-3

On2xl

Larger desktops (≥ 1400px)

Margin.On2xl.Is5
xxl-m-5

Responsive Chaining Examples

Chain breakpoints and directions (e.g. OnBase.Is4.OnLg).

Chain different margin sizes for different device types to create adaptive layouts.

Mobile to Laptop

Small margin on mobile, larger on laptop

Margin.OnBase.Is3.OnLg.Is4
sm-m-3 lg-m-4

Progressive Scaling

Increasing margins across device sizes

Margin.OnBase.Is2.OnMd.Is3.OnLg.Is4
sm-m-2 md-m-3 lg-m-4

Directional Responsive

Different margins for different sides on different devices

Margin.OnBase.Is3.FromTop.OnLg.Is4.FromBottom
sm-mt-3 lg-mb-4

Axis Responsive

Horizontal on mobile, vertical on laptop

Margin.OnBase.Is2.OnX.OnLg.Is3.OnY
sm-mx-2 lg-my-3

Phone to Wide Screen

Minimal on phone, generous on wide screens

Margin.OnBase.Is1.On2xl.Is5
m-1 xxl-m-5

Laptop to Desktop

Medium on laptop, large on desktop

Margin.OnLg.Is2.OnXl.Is4
lg-m-2 xl-m-4

Custom CSS and String Usage

Fluent API, CSS string, or class string for Margin.

You can use both the fluent API and custom strings for maximum flexibility.

Custom CSS String

Direct CSS style string

Margin="margin-top: 2rem; margin-bottom: 1rem;"

Custom CSS Class

Direct CSS class string

Margin="mt-4 mb-2"

Fluent API

Using the fluent API

Margin="Margin.Is3.FromTop.Is2.FromBottom"

Mixed Usage

Fluent API with custom CSS

Margin="Margin.OnBase.Is2" + Style="margin-top: 3rem;"
