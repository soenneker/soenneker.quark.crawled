Button

Displays a button or a component that looks like a button.

Size

Use the size parameter to change the size of the button.

Default

The default button is the primary action.

Outline

Outline buttons provide a lighter emphasis level for secondary actions.

Secondary

Secondary buttons provide an alternate filled emphasis level.

Ghost

Ghost buttons stay quiet until hover or focus.

Destructive

Use destructive buttons for irreversible actions.

Link

Link buttons keep button semantics while adopting link styling.

Icon

Icon buttons keep the same sizing and accessibility hooks as the rest of the button set.

With Icon

Remember to add the data-icon attribute to the icon for the correct spacing.

Rounded

Use the rounded-full class to make the button rounded.

Spinner

Render a Spinner component inside the button to show a loading state. Remember to add the data-icon attribute to the spinner for the correct spacing.

Button Group

To create a button group, use the ButtonGroup component.

As Child

You can use the AsChild prop on Button to make another component look like a button.
Login

RTL

Buttons preserve icon and text placement in right-to-left layouts.
