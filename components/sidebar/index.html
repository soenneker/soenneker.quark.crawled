Sidebar

A composable, themeable and customizable sidebar component.

Sidebars are one of the most complex components to build. They are central to any application and often contain a lot of moving parts.

We now have a solid foundation to build on top of. Composable. Themeable. Customizable.

Browse the Blocks Library.

Composition

Use the following structure to build a sidebar layout.

  • `SidebarProvider` owns the shared open, mobile, and keyboard-shortcut state.
  • `Sidebar`, `SidebarHeader`, `SidebarContent`, and `SidebarFooter` form the actual shell.
  • `SidebarMenuButton`, `SidebarMenuAction`, `SidebarMenuBadge`, and `SidebarMenuSub` cover the menu composition.
  • `SidebarInset` and `SidebarTrigger` keep the main content area wired to the same provider.

This page exercises the core menu composition: button-as-child links, group action, menu action, badge, submenu, footer, and rail.
