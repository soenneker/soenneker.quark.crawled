Accessibility Benefits

Using Fieldset and Legend for WCAG compliance.
Using Fieldset and Legend:
  • Groups related form controls logically
  • The legend provides a clear label for the group
  • Screen readers announce the legend When entering the fieldset
  • Required for WCAG compliance with grouped form controls
  • Disabling a fieldset disables all controls within it
  • Especially important for radio button and checkbox groups
