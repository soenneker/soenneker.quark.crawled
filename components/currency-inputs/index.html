Currency Input

Basic CurrencyInput

The default experience formats a nullable decimal value as currency while editing stays numeric.
$
Bound value: $1,234.56

Culture And Symbol Placement

You can keep the same decimal value while changing separators, decimal precision, and whether the symbol leads or trails the number.
$
Value: $2,499.95
EUR
Value: 1 999,95 EUR

Constraints

Min, max, negative handling, and custom precision help keep pricing and totals within safe business rules.
$
Clamped between 0 and 50,000: $15,000.00
credits
High precision value: 0.0425 credits

Validation

CurrencyInput participates in the existing validation primitives so totals and deposits can share the same field and error patterns.
