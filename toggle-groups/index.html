Toggle Group

A set of two-state buttons that can be toggled on or off.

Outline

Use the outline variant for grouped filters.

Size

Use the size parameter to scale every item in the group.

Spacing

Use spacing to separate grouped actions without changing their shared semantics.

Vertical

Use a vertical orientation when grouped actions need to fit into narrow side rails or inspector panels.

Disabled

Disable the entire group or individual items when a formatting choice is unavailable.

Custom

A custom toggle group example.

Use font-normal to set the font weight.

RTL

Toggle groups should preserve selection flow in right-to-left layouts.
