Responsive Layout

Vertical fields: Default orientation stacks label, control, and helper text—ideal for mobile-first layouts. Horizontal fields: Set orientation=&quot;horizontal&quot; on Field to align the label and control side-by-side. Pair with FieldContent to keep descriptions aligned. Responsive fields: Set orientation=&quot;responsive&quot; for automatic column layouts inside container-aware parents. Apply container-aware classes on FieldGroup to switch orientations at specific breakpoints.
