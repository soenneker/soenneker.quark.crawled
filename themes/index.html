Themes

Themes

Generate Quark theme CSS at compile time from a strongly typed Theme definition.

Install generator

Theme generation lives in the Soenneker.Quark.Gen.Themes package. Reference it as a build-time dependency in the app that owns the theme. 

																dotnet add package Soenneker.Quark.Gen.Themes

Create a theme

Add GenerateQuarkThemeCss to a class with exactly one public static method or property returning Theme. The factory can optionally take IServiceProvider. 

																
																	using Soenneker.Quark; using Soenneker.Quark.Gen.Themes; using Soenneker.Quark.Tokens;  [GenerateQuarkThemeCss("wwwroot/css/quark-theme.css")] public static class AppTheme {     public static Theme Build() => new()     {         Name = "App",         Tokens = new ThemeTokens         {             Light =             {                 Primary = "oklch(0.623 0.214 259.815)",                 PrimaryForeground = "oklch(0.985 0 0)"             },             Dark =             {                 Primary = "oklch(0.809 0.105 251.813)",                 PrimaryForeground = "oklch(0.205 0 0)"             }         },         Anchors = new AnchorOptions         {             Selector = "a",             DecorationLine = DecorationLine.None,             TextColor = "#2563ba"         },         Buttons = new ButtonOptions         {             BackgroundColor = "#2563eb",             TextColor = "#ffffff",             Rounded = "0.5rem",             Padding = "0.5rem 1rem"         },         Cards = new CardOptions         {             BackgroundColor = "#ffffff",             Rounded = "0.75rem",             BoxShadow = "0 0.25rem 0.75rem rgba(0,0,0,0.08)"         }     }; }

Use the emitted CSS

The generator writes the requested CSS file and, by default, a minified sibling. Reference the generated stylesheet after Quark's base styles. 

																
																	<link rel="stylesheet" href="_content/Soenneker.Quark.Suite/css/quark-tailwind.css" /> <link rel="stylesheet" href="css/quark-theme.css" />

Tailwind theme output

The same Theme also emits tailwind/quark-theme.generated.css unless TailwindThemeOutputFilePath is overridden. Import that file into the Tailwind input when the app should replace the baked-in shadcn theme variables during the Tailwind build. 

																@import "./quark-theme.generated.css";

Generated options

BuildUnminified, BuildMinified, and TailwindThemeOutputFilePath can be set on the attribute when an app needs different output behavior. 

																
																	[GenerateQuarkThemeCss(     "wwwroot/css/quark-theme.css",     BuildUnminified = true,     BuildMinified = true,     TailwindThemeOutputFilePath = "tailwind/quark-theme.generated.css")]
