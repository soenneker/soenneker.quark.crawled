Stack

`Stack` is a dedicated flex layout wrapper for one-dimensional flows. It defaults to a vertical stack, keeps the API obvious for the common cases, and still lets you layer on the broader Quark element and flex utility surface when a layout needs more control.

Default Vertical Stack

Items flow vertically out of the box.

The default orientation is vertical, which makes `Stack` a good fit for settings sections, forms, and any content flow where each child should sit underneath the previous one.

Horizontal Stack

Switch to a row-oriented layout for inline groups and toolbars.

Set `Orientation` to `Horizontal` when you want pills, actions, or summary cards to flow in a row.
